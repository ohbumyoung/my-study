1. 클래스와 객체의 정의
 
클래스: 객체를 만들기 위한 설계도, 틀이다. 
- 객체의 데이터와 기능(동작)을 정의한다.

객체: 클래스를 기반으로 생성된 인스턴스, 결과물이다.
-객체는 new 연산자와 생성자를 통해 생성되며, 실제 데이터는 메모리의 Heap 영역에 할당된다.
